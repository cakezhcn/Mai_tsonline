# Mai_tsonline

## 项目简介

Mai_tsonline 是一个让maimai查询ts3服务器的插件


## 安装方法
1.clone项目
```bash
git clone https://github.com/cakezhcn/Mai_tsonline.git
```
2.将其放在plugins下
3.启动Maicore生成config.toml，并且输入ts服务器的sq用户密码
4.重新启动


## 使用方法

可以使用```command```方法：
```
/ts|teamspeak
```
也可使用```action```,直接询问即可
